# P&L Calendar

A trading performance dashboard for tracking daily profit/loss, monitoring goals, and visualizing trading metrics with an intuitive calendar interface and real-time data visualization.

## About
- Passion project
- Used other apps and they all have ads so I decided to build my own for fun
- Made for trading but could really be used for anything (business, side hustle, gigs, etc) -- maybe i'll add multi calendars or smth

## Tech Stack

- **Frontend**: React, TypeScript, TailwindCSS, shadcn/ui, Recharts
- **Backend**: Supabase (Authentication & REST API)
- **Database**: Supabase/PostgreSQL
